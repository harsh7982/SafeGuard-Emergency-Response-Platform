# SafeHer – AI-Powered Women Safety Platform

## Overview

SafeHer is a cybersecurity-focused women safety platform designed to provide rapid emergency assistance, incident reporting, and trusted-contact notifications through a secure and scalable backend architecture.

The project is built using Spring Boot, PostgreSQL, JWT Authentication, and Docker, following modern backend development practices.

This project was developed as a final-year Computer Science and Cyber Security project with a focus on real-world safety applications.

---

## Key Features

### Secure Authentication

* JWT-based authentication and authorization
* Password encryption using Spring Security
* Role-based access control
* Protected API endpoints

### Emergency SOS System

* One-click SOS activation
* Automatic incident generation
* Location tracking support
* Emergency contact notification system

### Incident Management

* Report harassment, stalking, abuse, and emergency incidents
* Store incident history

* Investigation workflow support

### Emergency Contact Management

### Security Features

* Stateless authentication

### Deployment Ready

* Docker support
